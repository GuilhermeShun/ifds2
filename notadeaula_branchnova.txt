Pré-requisito:
- Ter um repositório git local

Instruções
Criação da branch nova
1. Abrir o terminal
2. Acessar o repositório local de interesse com o comando "cd"
3. Verificar a branch atual com o comando "git branch" 
4. Criar uma branch nova com o comando "git branch [nome da branch nova]"
5. Acessar a branch nova com o comando "git checkout [nome da branch nova]"
6. Trabalhar na branch nova, criando arquivos, etc

Envio para repositório remoto 
1. Realizar os procedimentos padrão de laboratório (git config --global user.name [nome], etc)
2. Definir o repositório remoto com "git remote add origin 'https://github.com/NomeDeUsuario/RepositorioRemoto.git'"
3. Obter o token no GitHub em https://github.com/settings/tokens/new, habilitando a opção "repo". 
4. Fazer o push do repositório local via terminal com o "git push -u origin [branch nova]", inserindo nome de usuário e token de verificação
